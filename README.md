# 🎨 ColorSym — Universal Color Symbols for Colorblind Accessibility

**ColorSym** is a free and open symbol library designed to help people with color vision deficiencies identify colors quickly and reliably. Whether you're designing a board game, website, interface, print material, or signage, ColorSym gives you a consistent, compact way to convey color information — no matter the context.

Originally created for use in board games, ColorSym is built for **universal accessibility** and **creative freedom**.

---

## ✨ Features

- Clear, intuitive glyphs for common colors
- Designed with accessibility and legibility in mind
- Available as a vector library **and** as a font
- Free for **personal and commercial use**
- Built for print and digital environments
- Easily embeddable in games, apps, UI, signage, and more

---

## 📦 What's Included

- `/svg/` folder — individual vector symbols (SVG format)
- `/font/` folder — font files (.otf, .ttf, .woff, etc.)
- `/docs/` — symbol reference guide and usage examples
- `ColorSym.css` — optional CSS class mapping for web projects
- `README.md` — you’re reading it!
- `LICENSE.txt` — license info for symbols and fonts

---

## 📘 Usage

You can use ColorSym symbols:

- 🃏  In **board and card games**
- 💻  In **software or app interfaces**
- 🧾  In **printed materials or packaging**
- 🚏  In **signage and navigation systems**
- 🎓  In **educational content and books**
- 🛒  In **commercial products and branding**

As long as you give proper attribution (see below), you may use the symbols and fonts in both personal and commercial projects.
